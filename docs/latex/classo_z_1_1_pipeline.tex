Callback handler that stores a bounded module\textquotesingle{}s callback function. 

If the callback returns false, the \mbox{\hyperlink{classo_z_1_1_pipeline}{Pipeline}} will not continue to process the current context state. It means that it will skip other modules. This can be used to implement caching in the pipeline. You can implement your own caching Module and process directly the current context to restore its cached version.
